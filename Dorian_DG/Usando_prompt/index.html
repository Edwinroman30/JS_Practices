Sintaxis de JavaScript prompt

prompt("Texto descriptivo","Texto por defecto");

Repasemos los parámetros:

Texto descriptivo: Es el mensaje que se mostrará por pantalla pidiendo la inserción o rellanado de información. Texto por defecto (Opcional): Podemos dejar un valor por defecto para que el usuario no tenga que escribirlo o rellenarlo. Con esta sintaxis y parámetros rellenados mostraremos por pantalla un mensaje acompañado de los botones ‘Aceptar’ y ‘Cancelar’. Además al usuario se le pide la introducción de texto.

Los valores devueltos serán ‘el valor introducido’ en caso de aceptar y ‘null’ en caso de cancelar.
